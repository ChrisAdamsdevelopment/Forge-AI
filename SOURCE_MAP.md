# Source Map

| Area | Folder | Role |
|---|---|---|
| Project control | `docs/` | Defines what Forge is, how it should behave, and what to build first |
| Prompt control | `prompts/` | Runtime prompt, project priming prompt, routing prompt, critique prompt |
| Tooling | `runtime/` | Ollama, MCP, Docker, and environment templates |
| RAG | `rag/` | Knowledge-base folder structure for future documents |
| Modules | `modules/` | Tasker-style automation/plugin system specs and examples |
| Training | `training/` | Fine-tuning notes and seed JSONL examples |
| Evaluation | `eval/` | Golden tasks and expected output examples |
| Implementation | `implementation/` | Initial code scaffold for the future app |
