# Forge Glossary

## Agent

A system that can reason over a task, call tools, inspect results, and continue until it reaches a final answer.

## MCP

Model Context Protocol. A standard way for AI applications to connect to tools and data sources.

## Root

A filesystem boundary that defines where a tool is allowed to operate.

## RAG

Retrieval-Augmented Generation. A method where relevant documents are retrieved and inserted into model context.

## Module

A portable Forge automation package containing prompts, schemas, tool permissions, examples, tests, and metadata.

## Tool policy

A rules file defining which tools may run automatically, which require approval, and which are denied.

## Golden task

A test prompt with expected behavior used to evaluate model, prompt, RAG, or tool changes.

## Adapter

A small fine-tuned model overlay such as LoRA or QLoRA weights.
