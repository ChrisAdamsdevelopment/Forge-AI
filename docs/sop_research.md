# SOP: Research Workflows

## Goal

Produce grounded, current, source-aware research that can guide implementation decisions.

## Steps

1. Restate the research question.
2. Identify what facts might be stale.
3. Search current sources when needed.
4. Prefer official docs, model cards, GitHub repos, and primary sources.
5. Build comparison tables.
6. Separate confirmed facts from inferences.
7. Identify missing variables.
8. Produce a decision recommendation.

## Required output

- Executive answer
- Comparison table
- Recommendation
- Risks
- Follow-up verification checklist
